ORAMICS

ORAMICS_back_to_LIFE

This project recreates the graphic synthesither Oramics, designed by Daphne Oram during the 60s, with a version for tabled/ipad or mobile using the open sources Puredata and Mobmuplat

Oramics was an instrument used as a compositional tool whose operation was based on translating graphical information into sound. It was conceived by Daphne Oram and built, together with several assistants, mainly between 1961 and 1972. This reinterpretation is framed within the composer‚Äôs own adaptations, which she designed to accommodate the technological changes of each era (Mini Oramics and Computer Oramics). Besides, this version of Oramics -ORAMICS_LIVE- was developed as a musical instrument for live performances and improvisation, rather than as a compositional tool

The project was presented as the Master‚Äôs Thesis in Electroacoustic Composition at Katarina Gurska in 2024. Oramics_Live was part of the Piksel 2024 Norway Festival in an online conference I have been using this version of Oramics for life performances and improvisation with n0n0is, being one of the main instruments in our first album LISTEN!üéß

ORAMICS_LIVE preserves the drawing system applied to most of the original features that determine the characteristics of the sound: four waveforms, four envelopes, vibrato, and pitch. However, since the purpose of the device has shifted from a compositional tool to an electronic instrument, some characteristics have been adapted and other have been added in this version. For example, reverberation is no longer determinated by a drawing system, insted, four different sliders have been integrated into the interface to provide more control over this parameter. In addition, a tunning system has also been developed for this version. Daphne Oram had already considered applying a tunning system in ther book An Individual Note (1971), even though there is no evidence that this element was ever integrated. With Oramics_LIVE it is possible to choose between a continous spectrum of frecuencies, or tune the instrument from quarter-tone to the octave intervals. Morever, it is also possible to determinate a maximum and minimum frecuency
